# Data Science Service (Anti-FOMO Stock Mentor)

Kalkulator indikator teknikal saham (RSI) berbasis Python FastAPI yang berfungsi sebagai mikroservis pendukung untuk *backend* utama Node.js.

## Tech Stack
- **Framework:** FastAPI
- **Data Processing:** Pandas
- **Technical Analysis:** `ta` library (Technical Analysis Library by Darb技术)
- **Server:** Uvicorn
